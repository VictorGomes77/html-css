Principais formatações de texto

Negrito / destaque

Nessa frase temos um termo em negrito usando a tag <b> tag b não tem semantica, ela é termo de forma.

Nesta frase temos um termo em destaque usando a tag <strong>, essa tag possui semântica, ela é um termo com significade de deixar o texto mais "forte".

Itálico / ênfase

Nesta frase, temos um termo itálico usando o tag <i>, ela não tem semântica, é um termo de forma.

Nesta frase, temos um termo em destaque usando a tag <em>, ela possui semântica.

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Para mudar a cor da marcação, é usado a linguagem CSS, nessa linguagem podemos estilizar o design.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também a tag sublinhado <u> mas ela não é semântica, apenas forma.

Texto sobrescrito

Para inserir coisas como x2 +3x + 4. Usando a tag <sup>

Texto subescrito

Para inserir coisas como H2O. Usando a tag <sub>
